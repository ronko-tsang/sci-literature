# Citation Tracing Guide

## The Problem

Review papers often cite other reviews, creating citation chains where:
```
Review A → Review B → Review C → Primary D
```

The original claim in Primary D may not support what Review A claims.

## The Solution: Trace to Primary

### Step-by-Step Protocol

1. **Identify the citation** in the review: "[X]"
2. **Go to reference list**: Find what "[X]" refers to
3. **Check the type**: Is it a primary study or another review?
4. **If review**: Continue tracing until you reach a primary study
5. **Verify**: Read the original to confirm it supports the claim

### Example

**Claim in Review A**: "Exercise reduces depression [15]"
**Tracing**:
- Ref [15] is → Review B (2020 review)
- Review B cites → Review C (2018 review)
- Review C cites → Primary D (RCT from 2015)

**Verification**: Read Primary D (RCT 2015)
- Does it actually show exercise reduces depression?
- Were the methods sound?
- What was the sample size?

**Correct citation**: Primary D (2015), NOT Review A/B/C

## When Tracing Is Required

| Claim Type | Must Trace? |
|------------|-------------|
| Specific quantitative finding | YES |
| Mechanistic explanation | YES |
| Clinical recommendation | YES |
| General background | NO |
| Framework/theory citation | NO |

## Evidence Strength Hierarchy

```
++ Strong: RCT, meta-analysis
+ Moderate: Cohort study, longitudinal
- Weak: Case-control, cross-sectional
? Poor: Expert opinion, non-peer-reviewed
```

## Citation Pollution Check

Count your citations:
- How many are primary studies?
- How many are reviews?
- Primary ratio should be > 80%

If reviews dominate:
- You have a citation pollution problem
- Apply SKILL 3 (trace to primary)
- Apply SKILL 5 (anti-pollution)
