# Evidence Audit Framework

## Purpose

Transform from passive reader to active evidence auditor.

## The Audit Protocol

For each major claim in a paper:

### 1. Source Identification

```
What type of source is this?
□ Primary research (original experiment)
□ Review (synthesizes other work)
□ Meta-analysis (statistical pooling)
□ Opinion/editorial
```

### 2. Evidence Location

```
Where does the evidence come from?
□ Direct experimental results (STRONG)
□ Cited reference (trace it)
□ Author interpretation (verify)
□ Unsupported assertion (FLAG)
```

### 3. Citation Chain Verification

```
Is the citation chain clean?
□ No intermediate reviews
□ Original study accessed
□ Methods verified
□ Sample size adequate
```

### 4. Consistency Check

```
Do other studies agree?
□ Consistent across multiple sources
□ Mixed results
□ Contradictory findings
□ Insufficient evidence
```

## Quality Signals

### Strong Evidence
- Large sample RCT
- Pre-registered protocol
- Peer-reviewed
- Multiple replications
- Effect size + confidence intervals

### Weak Evidence
- Small sample
- No control group
- Post-hoc analysis
- Industry funding
- No effect size reported

## Red Flags

Watch for:
- ❌ "Studies show..." with no citations
- ❌ Citations to review papers for specific claims
- ❌ Effect sizes without statistical significance
- ❌ Cherry-picked positive results
- ❌ Citation to non-peer-reviewed work
